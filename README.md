# reactor

Reactor is a simple React component that allows you to react on a component with an emoji and an optional comment.

## Demo

https://youtu.be/Db9t-3pCrQo

## Installation

- `npm i`

- `npm run dev`

```jsx
import Reactor from './components/Reactor'

<Reactor>
  {/* Your content goes here */}
</Reactor>
```
